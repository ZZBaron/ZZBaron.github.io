Poisson Process Simulation

View on GitHub

Project Overview

This simulation demonstrates various aspects of Poisson processes in one and two dimensions. The code generates visualizations of both homogeneous and inhomogeneous Poisson processes, showcasing their fundamental properties and applications in modeling random events.

Key Features

The simulation includes:

  • Visualization of Poisson distribution probability mass function
  • Homogeneous Poisson process simulation in 1D (time) with counting process
  • Analysis of exponential interarrival times in Poisson processes
  • Homogeneous spatial Poisson process simulation in 2D
  • Inhomogeneous Poisson process with time-varying intensity function
  • Inhomogeneous spatial Poisson process with position-dependent intensity

Mathematical Overview

Poisson Distribution

The Poisson distribution describes the probability of a given number of events occurring in a fixed interval, assuming events occur at a constant average rate and independently of each other.

Probability mass function (PMF):

\[ P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!} \]

where \(\lambda\) is the rate parameter (average number of events per interval) and \(k\) is the number of events.

Homogeneous Poisson Process

A stochastic process \(\{N(t), t \geq 0\}\) where:

  • \(N(0) = 0\)
  • The process has independent increments
  • The number of events in an interval of length \(t\) follows a Poisson distribution with mean \(\lambda t\):

\[ P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!} \]

Exponential Interarrival Times

In a Poisson process, the times between consecutive events (interarrival times) follow an exponential distribution:

Probability density function (PDF):

\[ f(t) = \lambda e^{-\lambda t} \]

Cumulative distribution function (CDF):

\[ F(t) = 1 - e^{-\lambda t} \]

Mean interarrival time: \(\mathbb{E}[T] = \frac{1}{\lambda}\)

Inhomogeneous Poisson Process

A generalization where the rate parameter \(\lambda\) varies with time: \(\lambda(t)\).

The expected number of events in an interval \([a,b]\):

\[ \mathbb{E}[N(b) - N(a)] = \int_a^b \lambda(t) dt \]

The probability of \(k\) events in interval \([a,b]\):

\[ P(N(b) - N(a) = k) = \frac{\left(\int_a^b \lambda(t) dt\right)^k e^{-\int_a^b \lambda(t) dt}}{k!} \]

Spatial Poisson Process (2D)

For a homogeneous spatial Poisson process with intensity \(\lambda\) (points per unit area), the number of points \(N(S)\) in a region \(S\) follows:

\[ P(N(S) = k) = \frac{(\lambda |S|)^k e^{-\lambda |S|}}{k!} \]

where \(|S|\) is the area of region \(S\).

Inhomogeneous Spatial Poisson Process

The intensity varies with location: \(\lambda(x,y)\).

The expected number of points in region \(S\):

\[ \mathbb{E}[N(S)] = \int\int_{S} \lambda(x,y) dx dy \]

The probability of \(k\) points in region \(S\):

\[ P(N(S) = k) = \frac{\left(\int\int_{S} \lambda(x,y) dx dy\right)^k e^{-\int\int_{S} \lambda(x,y) dx dy}}{k!} \]

Simulation Methods

In the code, two primary methods are used to simulate Poisson processes:

  1. Direct method: For homogeneous processes, generate the total number of points from a Poisson distribution, then uniformly distribute these points.
  2. Thinning/Rejection method: For inhomogeneous processes, generate points according to a homogeneous process with intensity equal to the maximum of the inhomogeneous intensity function, then randomly reject points with probability \(1 - \frac{\lambda(t)}{\lambda_{max}}\) or \(1 - \frac{\lambda(x,y)}{\lambda_{max}}\).

Applications

Poisson processes model many real-world phenomena including:

  • Arrivals of customers at a service point
  • Number of calls to a call center
  • Decay events in radioactive materials
  • Network packet arrivals
  • Spatial distribution of trees, stars, or other natural phenomena
  • Occurrence of rare diseases or accidents
