Hydrogen Wavefunction Eigenstates

View on GitHub

Project Overview

This project provides a comprehensive visualization of hydrogen atom quantum states by solving both analytical and numerical forms of the Schrödinger equation. It demonstrates the complex probability distributions of electron orbitals through interactive 3D visualizations and animated rotations of probability density planes.

Key Features

The simulation provides:

  • Comparative visualization of analytical and numerical solutions to the radial Schrödinger equation
  • Real-time 3D visualization of probability density distributions in rotatable planes
  • Interactive controls for rotation angles (θ and φ) to explore orbital geometry
  • Support for any valid combination of quantum numbers (n, l, m)
  • Animated orbital visualization showing both φ and θ rotations
  • Normalized probability density calculations with color-coded visualization

Technical Implementation

Built using Python with several specialized scientific computing libraries:

  • NumPy and SciPy for quantum mechanical calculations and differential equation solving
  • Matplotlib for 3D visualization and interactive plotting
  • Custom implementation of both analytical and numerical solutions for radial wavefunctions
  • Vectorized calculations for improved performance
  • Integration with spherical harmonics for angular components

Mathematical Foundation

The simulation solves the quantum mechanical hydrogen atom using multiple approaches:

  • Analytical solution using generalized Laguerre polynomials and spherical harmonics
  • Numerical solution of the radial Schrödinger equation using solve_ivp
  • Proper normalization of wavefunctions through numerical integration
  • Transformation between spherical and Cartesian coordinates for visualization
  • Complete treatment of quantum numbers n, l, and m with appropriate constraints
