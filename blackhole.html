Black Hole Light Ray Simulation

View on GitHub

Project Overview

This simulation explores the fascinating behavior of light rays near a black hole. Using principles from general relativity, it demonstrates how strong gravitational fields bend the path of light, creating unique visual phenomena.

Key Features

The simulation allows users to visualize: - Light ray trajectories near the event horizon - Formation of Einstein rings - Effects of gravitational lensing on light paths - Real-time manipulation of black hole parameters

Technical Implementation

Built using Python with NumPy for calculations and Matplotlib for visualization. The simulation solves the geodesic equations in curved spacetime to accurately model light behavior in strong gravitational fields.

Mathematical Background

The simulation is based on the mathematical framework of general relativity, specifically focusing on null geodesics in Schwarzschild spacetime. This allows for accurate representation of how light behaves in the extreme gravitational conditions near a black hole.
