2D N-Body Gravitational Simulation

View on GitHub

Project Overview

This simulation models the gravitational interactions between multiple bodies in a 2D space using newtonion mechanics.

Key Features

The simulation includes:

  • Customizable particle masses and initial conditions
  • Verlet velocity integration for accurate orbital mechanics
  • Continuous collision detection and resolution
  • Real-time and precalculated animation modes
  • Interactive timeline slider for reviewing particle paths
  • Energy conservation tracking

Technical Implementation

Developed in Python, the simulation leverages NumPy for efficient numerical computations and Matplotlib for visualization. The implementation features object-oriented design with separate classes for particles, simulation physics, and animation handling.

Mathematical Background

The simulation implements classical mechanics principles including Newton's law of universal gravitation and conservation of momentum. It uses the Verlet velocity integration method for stable orbital calculations and implements elastic collision physics with continuous detection to handle particle interactions accurately. A softening parameter is included to prevent numerical instabilities in close encounters.
