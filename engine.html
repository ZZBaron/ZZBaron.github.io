3D Physics Game Engine (WIP)

View on GitHub

Project Overview

A comprehensive 3D physics engine built using OpenGL and NVIDIA's PhysX SDK. The engine combines modern graphics rendering with industrial-strength physics simulation, providing a robust platform for real-time physics-based applications and games.

Key Features

  • Integration with NVIDIA PhysX for physics simulation
  • Advanced material system with physically based shaders
  • Real-time shadow mapping with PCF filtering for soft shadows
  • Support for primitive shapes (spheres, boxes, cylinders)
  • Interactive camera system with configurable movement modes
  • ImGui-based debug interface for real-time parameter adjustment
  • Scene management with object hierarchy support
  • Advanced asset importing with texture and material handling

Technical Implementation

Built using modern C++20 and OpenGL, featuring:

  • CMake-based build system with vcpkg package management
  • Modern shader-based rendering pipeline
  • Sophisticated material system supporting PBR workflows
  • Scene graph architecture for complex object hierarchies
  • Cross-platform compatibility (Windows, Linux, macOS)
  • Assimp integration for 3D model importing (fbx, obj, gltf)
  • Robust file I/O system for asset management

Physics Features

Powered by NVIDIA PhysX, the engine supports:

  • Dynamic and static rigid bodies with full 6-DOF simulation
  • Efficient broad-phase and narrow-phase collision detection
  • Material-based physical properties (friction, restitution)
  • Real-time physics debugging and visualization
  • Configurable gravity and environmental parameters

Graphics Features

  • Dynamic shadow mapping with configurable resolution
  • Support for multiple material types and textures
  • Texture uv mapping
  • Wireframe visualization mode for debugging
  • Configurable lighting with adjustable parameters

Development Tools

  • Interactive ImGui-based debug interface
  • Real-time physics parameter adjustment
  • Performance monitoring and debugging tools
  • Scene object hierarchy viewer
  • Material and texture preview system
