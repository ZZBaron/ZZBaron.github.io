3D Physics Game Engine (WIP)

View on GitHub

Project Overview

A real-time 3D physics engine built from scratch using OpenGL and C++. The engine features a robust rigid body dynamics system with accurate collision detection and response, supporting both real-time physics simulation and pre-computed physics playback modes.

Key Features

The engine provides:

  • Rigid body dynamics with 6 degrees of freedom (translation and rotation)
  • Advanced collision detection using SAT (Separating Axis Theorem) and convex hull generation
  • RK4 numerical integration for accurate physics simulation
  • Real-time and pre-computed simulation modes with interpolated playback
  • Dynamic shadow mapping and modern rendering techniques
  • Support for various 3D primitives including spheres and rectangular prisms
  • Texture mapping and material system with configurable properties

Technical Implementation

Built using modern C++ and OpenGL, the engine implements:

  • Custom rigid body physics solver with quaternion-based rotations
  • AABB-based broad-phase and SAT-based narrow-phase collision detection
  • Impulse-based collision resolution with configurable restitution
  • Shader-based rendering pipeline with shadow mapping and texture support
  • Interactive camera system with configurable views and tracking modes
  • Efficient memory management using smart pointers and optimized data structures

Physical Systems

The engine accurately simulates:

  • Gravitational forces with configurable field strength
  • Angular momentum and inertia tensor calculations
  • Energy preservation and momentum conservation in collisions
  • Air resistance and rotational damping effects
  • Static and dynamic friction (configurable coefficients)
  • Resting contact resolution with stability corrections
